# DogCore

A drop-in replacement for Essentials and other small plugins. Used on [Dumb Dog Diner](https://dumbdogdiner.net/)

## Features

- Teleportation commands such as /home, /tpa, /spawn, and /back
- Nicknames
- Permission group colors and prefixes
- Auto-AFK
- Economy
- And more!
